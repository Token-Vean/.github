# Suite local para procesamiento documental

Suite de herramientas locales y de código abierto para apoyo al procesamiento documental, análisis archivístico, revisión de riesgos, anonimización y asistencia técnica en entornos de gestión documental.

Los proyectos priorizan ejecución local, privacidad, trazabilidad, validación profesional y uso responsable de inteligencia artificial.

## Principios de diseño

- Ejecución local siempre que sea posible.
- Minimización de envío de datos a servicios externos.
- Trazabilidad de decisiones y resultados.
- Validación profesional obligatoria.
- Transparencia sobre límites, riesgos y dependencias.
- Utilidad práctica para archivos, bibliotecas, centros de documentación y unidades de gestión documental.
